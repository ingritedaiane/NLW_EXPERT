 ## 💻 Projeto

sobre a linguagem de programação JavaScript<br>
As perguntas foram criadas utilizando a ajuda de uma IA ( ChatGPT ) e o projeto foi estruturado com HTML, CSS e JavaScript, deploy com Github Pages.<br>
